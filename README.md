# terminal.me

> A personal developer portfolio — minimal, monochromatic, and built with a terminal soul.

🔗 **Live:** [[custom-name].vercel.app](https://[custom-name].vercel.app) ← update after deploying

---

## Stack

```
Next.js 14 (App Router)   →  framework
TypeScript                →  language
Tailwind CSS              →  styling
Framer Motion             →  animations
IBM Plex Mono             →  font
Vercel                    →  deployment
```

---

## Run Locally

```bash
git clone https://github.com/prthm412/terminal.me.git
cd terminal.me
npm install
npm run dev
```

Open [localhost:3000](http://localhost:3000).

---

## Structure

```
app/            → pages and layout
components/     → UI and section components
data/           → content (projects, skills, academics)
public/         → static assets
```
